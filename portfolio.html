Summary

This web application allows users to enter the URL of a YouTube Playlist, to retrieve detailed information not currently found on YouTube.

One such detail is the total Playlist length, which is the reason I first started the project. I watch a lot of programming tutorials on YouTube and was curious how long each would take to watch.
