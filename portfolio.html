Youtube Playlist Details

Web Application

HTML, CSS, JavaScript, jQuery, Responsive Design, GitHub, Brackets.

Youtube Playlist Details uses three YouTube API’s to gather details about a YouTube playlist (via a user provided URL).

It provides some details not shown on YouTube (or via the API’s) such as: the total length of a playlist (i.e. the sum of all the video lengths).
